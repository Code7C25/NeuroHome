NeuroHome

NeuroHome es una aplicación móvil conectada a un sistema de automatización con Arduino y ESP32 que permite controlar dispositivos del hogar, como puertas y sensores, a través de la app. La idea es ofrecer un hogar inteligente accesible y seguro, integrando hardware y software de manera sencilla.

Problema y solución

Problema:
Las casas tradicionales no permiten un control remoto eficiente de puertas y sistemas de seguridad, y los sistemas existentes suelen ser costosos o difíciles de configurar.

Solución:
NeuroHome combina una aplicación móvil con hardware Arduino/ESP32 para ofrecer un sistema de control de puertas y sensores del hogar. Los usuarios pueden abrir puertas, monitorear sensores y recibir notificaciones desde cualquier lugar de manera sencilla y segura.

Tecnologías utilizadas

Lenguajes de programación: Dart (Flutter), C++ (Arduino IDE)

Frameworks: Flutter

Base de datos: MySQL

Herramientas: VS Code, Arduino IDE, GitHub, Trello

Cómo usar el sistema:

Instalar la aplicación NeuroHome en Android (y próximamente iOS).

Conectar el Arduino UNO con ESP32 al sistema eléctrico y sensores del hogar.

Ingresar a la app y registrarse o iniciar sesión.

Controlar puertas, ventanas, garaje, luces y sensores desde la app.
